How to solve “remote host identification has changed”?

Posted on 2021-03-31 Edited on 2021-05-14

Problem

Met this warning when I tried to connect to AWS Lightsail using ssh -i key.pem username@ipAddress

Solution

The warning can be generated by many reasons but 99% of the times, Operating system is re-installed on the remote server that will change its identity, and your local server has an old identity of your remote server.

To solve the issue, Just execute user@locahost:~$ ssh-keygen -R REMOTE_HOST on client-server.
This command remove the old remote host identity from the known_hosts file.

  1. https://serveravatar.com/remote-host-identification-has-changed/
